# Data files needed for practice

This folder contains GitHub-readable transcriptions of the 2021-2025 BAN404 exam papers.

## Files you can already use from this folder

- `airline.csv` — needed for the 2024 exam and copied here from `Previous Exam/2024/airline.csv`.
- `Churn.csv` — needed for the 2023 exam and copied here from `Previous Exam/2023/Churn.csv`.

## File you still need to add

- `data_task1.csv` — needed for Task 1 in the 2025 exam. The exam paper says this file was attached, but it is not present in the repository right now.

## Exams that do not need an extra local data file

- **2022**: uses package datasets (`OJ` from `ISLR`, `Computers` from `Ecdat`).
- **2021**: uses package datasets (`Smarket` from `ISLR`, `dataCar` from `insuranceData`).

## Practical note

If you want this folder to be fully runnable as a single practice folder, the only missing local data file is `data_task1.csv` for 2025.
