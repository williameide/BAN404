About the exam

Some comments about the exam:

Concerning Deep Learning: During the lecture about this topic (which could have been named "Neural Networks" since only simple versions were considered) I used functions such as MLPRegressor, from the sklearn.neural_network library, rather than more advanced approach, using torch and some other libraries, taken by the book. Also, I did not treat convolutional, nor recurrent, neural networks and you should not expect to be asked about that on the exam.
On descriptive statistics: When this is asked for, the choices of plots and tables to use are context dependent, and are often part of the question. An example, which has occurred on previous exams is something along the lines of "Use descriptive statistics to find promising predictors". In this case some possible methods to use are scatterplots of continuous responses and a continuous predictors, boxplots for continuous responses for different values of a categorical predictor, boxplots of a continuous predictor for different values of a categorical response or a cross tabulation of values for a categorical predictor and a categorical response (perhaps normalized so that each cell estimates P(response | predictor) ).
